# ☁️ Cloud Task Scheduling System – Intelligent Load Balancing Framework

## 📌 Overview
Cloud Task Scheduling System is a simulation-based project designed to optimize task execution in cloud computing environments. It uses a Decision Tree-based scheduling approach combined with dynamic load balancing techniques to efficiently allocate tasks to virtual machines. The system focuses on improving execution time, reducing energy consumption, and maximizing resource utilization in heterogeneous cloud environments.

---

## 🚀 Key Features
- 🌳 Decision Tree-based intelligent task scheduling  
- ⚖️ Dynamic load balancing using LSQ (Local Shortest Queue)  
- 🖥️ Virtual Machine (VM) based task allocation  
- ⚡ Reduced execution time (makespan)  
- 🔋 Energy-efficient scheduling mechanism  
- 📊 Real-time workload monitoring and redistribution  
- ☁️ Scalable cloud simulation using CloudSim  

---

## 🧠 System Architecture
The system follows a cloud simulation architecture consisting of:

- *Data Centers:* Simulated cloud infrastructure  
- *Brokers:* Manage task scheduling and resource allocation  
- *Virtual Machines (VMs):* Execute assigned tasks  
- *Users:* Submit computational tasks  

The scheduler uses Decision Tree logic and load balancing strategies to assign tasks efficiently across available resources.
